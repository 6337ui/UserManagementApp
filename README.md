# UserManagementApp

Веб-приложение для управления пользователями, разработанное как тестовое задание. Приложение предоставляет удобный интерфейс для просмотра, добавления и поиска пользователей.

## Технологический стек

- **Vue 3** – прогрессивный фреймворк для построения пользовательских интерфейсов
- **Pinia** – официальное решение для управления состоянием в Vue 3
- **Tailwind CSS** – utility-first фреймворк для стилизации
- **Vite** – современная сборка для быстрой разработки

## Структура проекта

```
src/
├── components/             # Компоненты интерфейса
│   ├── composables/        # Мелкие переиспользуемые компоненты
│   ├── header.vue          # Шапка приложения
│   ├── footer.vue          # Подвал приложения
│   └── table.vue           # Таблица пользователей
├── stores/                 # Хранилища состояния
│   └── visitorsStores.js   # Хранилище для управления данными пользователей
├── App.vue                 # Главный компонент приложения
└── main.js                 # Точка входа приложения
```

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите режим разработки:
```bash
npm run dev
```

3. Для сборки проекта в продакшен:
```bash
npm run build
```

4. Для предварительного просмотра продакшен-сборки:
```bash
npm run preview
```

Приложение будет доступно по адресу `http://localhost:5173`.

## Особенности

- Адаптивный интерфейс
- Управление состоянием через Pinia
- Быстрая сборка и горячая перезагрузка с помощью Vite
- Аккуратная и современная визуальная стилизация с Tailwind CSS
